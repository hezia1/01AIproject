# AI 原生应用安全检测、验证与治理平台 PRD

## 1. 项目定位

本项目面向软件供应链、复杂业务系统和 AI Agent 生态，建设一个具备业务理解、风险关联、动态验证和持续进化能力的应用安全平台。

平台不定位为单一扫描工具，而定位为企业级应用安全能力底座：

- 汇聚代码、依赖、配置、Agent 工具、漏洞、误报和整改数据。
- 用 AI 辅助安全人员完成漏洞发现、复核、解释和修复建议。
- 用动态验证和证据链降低误报，提高高危风险处置效率。
- 通过知识库、规则库和 Skill 库沉淀企业自己的安全能力。

## 2. 目标用户

- 安全工程师：执行扫描、复核漏洞、生成报告、推动整改。
- 安全负责人：查看风险趋势、项目风险排名、整改 SLA 和治理效果。
- 研发负责人：接收安全问题、查看影响范围、安排修复和复测。
- 平台管理员：管理用户、项目组、权限、扫描资源和系统配置。

## 3. 核心问题

传统安全工具存在以下问题：

- SAST、SCA、DAST 数据割裂，难以统一治理。
- 扫描结果误报多，安全人员消耗在低价值筛选上。
- 工具不理解业务语义，难发现逻辑漏洞、越权链路和复杂数据流风险。
- 供应链风险只停留在组件漏洞匹配，缺少影响分析和修复优先级。
- Agent、MCP、插件和工具调用引入新的攻击面，传统工具覆盖不足。
- 安全经验难沉淀，依赖个人专家能力。

## 4. 产品目标

第一阶段 MVP 目标：

- 支持创建项目并接入 Git 仓库。
- 支持解析主流依赖文件并生成 SBOM。
- 支持基础 SCA 风险识别和许可证风险识别。
- 支持基础 SAST 规则扫描。
- 支持 AI 对漏洞结果进行解释、复核和修复建议生成。
- 支持漏洞状态流转、复测记录和报告导出。

中长期目标：

- 建立代码图谱、依赖图谱、业务知识图谱和误报经验库。
- 支持静态发现到动态验证的三态裁决。
- 支持 Agent/MCP/插件安全检测。
- 支持多 Agent 协同审计流程。
- 支持私有化部署、行业规则包和企业定制 Skill。

## 5. MVP 功能范围

### 5.1 用户与权限

- 用户登录。
- 用户、角色、项目组管理。
- 角色包括管理员、安全工程师、研发用户、只读审计用户。

### 5.2 项目管理

- 创建项目。
- 配置项目名称、业务线、负责人、仓库地址、默认分支。
- 查看项目风险概览。
- 查看历史扫描任务。

### 5.3 仓库接入

- 支持 Git 仓库地址配置。
- 支持手动触发拉取和扫描。
- 记录代码版本、提交哈希、扫描时间。

### 5.4 SCA 供应链扫描

MVP 支持以下文件：

- `package.json`
- `package-lock.json`
- `requirements.txt`
- `pom.xml`
- `go.mod`

输出：

- 组件名称、版本、生态、直接/传递依赖标识。
- 漏洞编号、严重等级、影响版本、修复版本。
- 许可证名称和许可证风险等级。
- 修复优先级建议。

### 5.5 基础 SAST 扫描

MVP 先使用规则扫描，不做完整编译级程序分析。

优先覆盖：

- 硬编码密钥。
- SQL 注入危险调用。
- 命令执行危险调用。
- SSRF 危险调用。
- 路径穿越风险。
- 不安全反序列化。
- 弱加密算法。
- 敏感配置暴露。

### 5.6 AI 辅助复核

对高危和中危发现调用 AI 复核流程：

- 解释漏洞触发原因。
- 判断可能误报点。
- 生成影响说明。
- 生成修复建议。
- 生成面向研发的简洁描述。

MVP 阶段 AI 结论不直接自动关闭漏洞，只作为辅助信息。

### 5.7 漏洞治理

- 漏洞列表。
- 严重等级、类型、状态、负责人、项目、扫描来源筛选。
- 状态流转：待确认、已确认、修复中、待复测、已关闭、误报。
- 支持备注、复测记录和附件证据。

### 5.8 报告导出

- 项目风险摘要。
- 漏洞明细。
- 供应链组件清单。
- 修复建议。
- 支持 HTML 或 PDF 报告。

## 6. 暂不纳入 MVP 的能力

以下能力放入二期或三期：

- 完整业务知识图谱。
- 深度代码图谱和跨函数数据流。
- 自动 PoC 生成。
- DAST 自动验证。
- 沙箱运行时证据链。
- 多 Agent 编排控制台。
- Agent/MCP/插件专项扫描。
- CI/CD 安全门禁。
- 行业规则包市场。

## 7. 成功标准

MVP 完成后应满足：

- 能接入一个真实 Git 仓库并完成扫描。
- 能输出依赖清单、基础漏洞和许可证风险。
- 能发现至少 8 类基础代码风险。
- 能对高风险结果生成 AI 解释和修复建议。
- 能完成漏洞确认、修复、复测、关闭的闭环。
- 能导出项目级安全报告。
